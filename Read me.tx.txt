# Weather Forecast App

## Project Overview

The Weather Forecast App is a simple and interactive web application built using HTML, CSS, and JavaScript. It allows users to search for any city and view weather-related details such as temperature, weather condition, humidity, wind speed, pressure, and visibility.

This project is designed with a modern multi-page interface using smooth button navigation between pages. It is beginner-friendly and suitable for mini projects, college submissions, and frontend practice.

---

## Features

* Multi-page weather app design
* Search weather by city name
* Temperature display
* Weather condition with icons
* Humidity details
* Wind speed information
* Atmospheric pressure display
* Visibility range display
* Beautiful UI with glassmorphism effect
* Responsive and clean design
* Button-based page navigation

## How to Run the Project

1. Download or copy the project code
2. Save the file as `index.html`
3. Open the file using any web browser such as Chrome, Edge, or Firefox
4. Click on **Get Started**
5. Enter a city name
6. Click **Search Weather**
7. View the weather details on the next page

---

## How It Works

The application uses JavaScript to generate sample weather data dynamically when a city is searched.

It displays:

* Random temperature
* Random weather condition
* Random humidity
* Random wind speed
* Random pressure
* Random visibility

This makes the project visually interactive for demonstration purposes.
